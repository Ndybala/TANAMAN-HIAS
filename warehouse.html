Pengertian Data Warehouse

Data Warehouse adalah tempat penyimpanan data yang digunakan untuk mengumpulkan, mengintegrasikan, dan menganalisis data dari berbagai sumber. Pada sistem Ekosistem Bisnis Tanaman Hias, Data Warehouse digunakan untuk menyimpan data penjualan, stok, pelanggan, dan supplier yang nantinya dimanfaatkan dalam proses analisis bisnis dan pembuatan laporan.

Sumber Data

No Sumber Data Keterangan
1 Data Tanaman Informasi produk tanaman hias.
2 Data Transaksi Riwayat penjualan dan pembelian.
3 Data Supplier Informasi pemasok tanaman.
4 Data Pelanggan Informasi pelanggan.
5 Data Stok Jumlah persediaan tanaman.

Manfaat Data Warehouse

  • 📊 Menyediakan laporan penjualan yang akurat.
  • 🌿 Mengetahui tanaman paling laris.
  • 📦 Memantau stok tanaman.
  • 📈 Membantu analisis perkembangan bisnis.
  • 💰 Mengetahui keuntungan setiap periode.
  • 🎯 Membantu pengambilan keputusan bisnis.

Alur Data Warehouse

Tahapan Penjelasan
Extract Mengambil data dari database transaksi.
Transform Membersihkan dan mengolah data agar konsisten.
Load Menyimpan data ke dalam Data Warehouse.
Analysis Menganalisis data untuk menghasilkan laporan.
