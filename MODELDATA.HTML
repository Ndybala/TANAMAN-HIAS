Pengertian Model Data

Model data merupakan representasi struktur data yang digunakan dalam suatu sistem untuk menggambarkan hubungan antar data. Pada sistem Ekosistem Bisnis Tanaman Hias, model data digunakan sebagai dasar dalam merancang database sehingga seluruh informasi dapat dikelola dengan baik dan saling terhubung.

Entitas Utama

No Entitas Deskripsi
1 User Menyimpan data pengguna sistem.
2 Tanaman Menyimpan informasi tanaman hias.
3 Kategori Jenis atau kelompok tanaman.
4 Supplier Data pemasok tanaman.
5 Stok Informasi jumlah stok tanaman.
6 Transaksi Data penjualan tanaman.
7 Detail Transaksi Rincian setiap transaksi.
8 Pembayaran Informasi pembayaran pelanggan.

Hubungan Antar Entitas

Entitas Berhubungan Dengan Jenis Relasi
User Transaksi One to Many
Supplier Tanaman One to Many
Tanaman Stok One to One
Transaksi Detail Transaksi One to Many
Detail Transaksi Pembayaran One to One

Manfaat Model Data

  • ✅ Mempermudah perancangan database.
  • ✅ Mengurangi terjadinya redundansi data.
  • ✅ Memudahkan proses pengembangan sistem.
  • ✅ Menjaga konsistensi data.
  • ✅ Mempermudah pembuatan ERD dan normalisasi.
